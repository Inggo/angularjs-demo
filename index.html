What is needed?

  1. AngularJS
    • https://angularjs.org/
    • 1.3x (stable)
    • Zip

  2. Local web server (Xampp, Wamp, Mamp)
    • For HTTP Requests only

  3. Bootstrap and Jquery (optional)
    • http://getbootstrap.com/getting-started/#download
    • https://jquery.com/download/

Initial Setup

  1. Create a project folder.
  2. Create a "libs" folder inside.
    • Paste "angular-1.3.15" folder.
    • Paste "bootstrap-3.3.4" folder.
    • Paste "jquery-1.11.2.min.js"
  3. Create index.html
    • http://getbootstrap.com/getting-started/#template
    • Copy-paste
    • Edit the folder and file names
